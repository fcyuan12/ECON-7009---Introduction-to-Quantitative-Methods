Introduction to Quantitative Methods

Question 6

Discuss why the following proofs are incorrect.

  1. (5%) Consider \(f: S \subset \mathbb{R} \rightarrow \mathbb{R}\), where \(S\) is an open set in \(\mathbb{R}\). Assume that \(f\) is continuous at a point \(c\) in \(S\) and that \(f(c) > 0\). Prove that there exists an open set \(U \subset S\), such that for all \(x \in U\), \(f(x) > 0\).

Proof. Because \(f(c)\) is an interior point of \((0, \infty)\), there exists a \(\delta > 0\) such that \[\begin{align} B\left(f(c), \delta \right) \subset (0, \infty). \end{align}\] Because \(B\left(f(c), \delta \right)\) is an open set in \(\mathbb{R}\), its inverse image \(f^{-1}\left( B\left(f(c), \delta\right) \right)\) is open in \(S\). Therefore, we find an open set \(U = f^{-1}\left( B\left(f(c), \delta \right) \right)\) in \(S\) such that for all \(x \in f^{-1}\left( B\left(f(c), \delta \right) \right)\), \[\begin{align} f(x) \in B\left(f(c), \delta \right) \subset (0, \infty), \end{align}\] implying that \(f(x)>0\).

  1. (5%) Prove that if \(S_1\) and \(S_2\) are compact, then \(S_1+S_2\) is compact.

Proof. Pick any sequence \(\{ z_n \}\) and \(z_n \in S_1 + S_2\) for all \(n\). We want to show that we can find a convergent subsequence of \(\{ z_n \}\) such that its limit \(z\) is in \(S_1 + S_2\). Because \(z_n \in S_1+S_2\), this implies that we can find a point \(x_n \in S_1\) and \(y_n \in S_2\) such that \(z_n = x_n + y_n\). Because \(S_1\) is compact, and \(\{x_n\}\) is a sequence in \(S_1\), we can find a convergent subsequence of \(\{x_n\}\), denoted by \(\{x_{k(n)} \}\) such that \[\begin{align} \lim_{n \rightarrow \infty} x_{k(n)}=x \in S_1. \end{align}\] Because \(S_2\) is compact, and \(\{ y_n \}\) is a sequence in \(S_2\), this implies that its subsequence \(\{y_{k(n)} \}\) converges to a point \(y \in S_2\). Therefore, we find a subsequence \(\{z_{k(n)} \}\) converges to \(z = x + y\), and \(z \in S_1+S_2\), which completes that proof that \(S_1+S_2\) is compact.

(Rudin and others 1964, 3:86 Theorem 4.8)

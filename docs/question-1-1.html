Introduction to Quantitative Methods

Question 1

Let \(S=\left\{ x\in \mathbb{R} \mid x= 1- \frac{1}{n}, \ n \in \mathbb{Z}^{+} \right\} \cup \left\{1 \right\}.\)

  1. (6%) Is \(S\) closed?

  2. (6%) Is it compact?

  3. (6%) Is it convex?

Related Definition or Theorem

Definition 1 A set \(E \in \mathbb{R}^{k}\) is convex if \[\begin{align} \lambda x + (1 - \lambda) y \in E \end{align}\] where \(x\in E, y\in E,\) and \(\lambda \in [0,1]\).

(Rudin and others 1964, 3:31)

Definition 2 Let \(X\) be a metric space. \(E\) is bounded if there is a real number \(M\) and a point \(q\in X\) such that \(d\left( p,q \right) < M\) for all \(p\in E\).

(Rudin and others 1964, 3:34 Definition 2.18 (i))

Theorem 1 A set \(S\) is open if and only if its complement \(S^C\) is closed.

(Rudin and others 1964, 3:34 Theorem 2.23)

Theorem 2 For any collection \(\left\{ G_\alpha \right\}\) of open sets, \(\bigcup_\alpha G_\alpha\) is open.

(Rudin and others 1964, 3:34 Theorem 2.24 (i))

Theorem 3 (Heine-Borel Theorem) A set \(A \subset \mathbb{R}^n\) is compact if and only if it is closed and bounded.

(Marsden, Hoffman, and others 1993, 155 Theorem 3.2.1)

Solution (a)

We know \[\begin{align} S & = \left\{ x\in \mathbb{R} \mid x = 1 - \frac{1}{n}, n\in \mathbb{Z}^+ \right\} \cup \left\{1 \right\} \\ & = \left\{ 0, \frac{1}{2}, \frac{2}{3}, \frac{3}{4}, \ldots, 1 \right\}. \end{align}\] (2%) Then, \[\begin{align} S^{C} & = \left( -\infty, 0 \right) \cup T \cup (1,\infty). \end{align}\] where \[\begin{align} T & = \bigcup_{n\in \mathbb{N}} \left( 1-\frac{1}{n}, 1-\frac{1}{n+1} \right). \end{align}\] (2%) By Theorem 2, \(S^C\) is open.

(2%) By Theorem 1, \(S\) is closed.

Solution (b)

(2%) Take \(r = 2\).

(2%) Then, \[\begin{align} S \subset B(0, r). \end{align}\] That is, \(S\) is bounded by Definition 2.

(2%) By Theorem 3, \(S\) is compact.

Solution (c)

(3%) Let \[\begin{align} x_1 & = 0 \in S \\ x_2 & = \frac{1}{2} \in S \\ \theta & = \frac{1}{2} \in [0,1]. \end{align}\] (3%) Then, \[\begin{align} \theta x_1 + \left( 1-\theta \right) x_2 & = \frac{1}{2}\cdot 0 + \left( 1-\frac{1}{2} \right) \cdot \frac{1}{2} \\ & = \frac{1}{4} \notin S. \end{align}\] Thus, \(S\) is not convex by Definition 1.
